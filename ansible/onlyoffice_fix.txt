 Installation manuelle (recommandée)
Cette méthode est la plus fiable, surtout si tu utilises une connexion avec un proxy ou si l'accès à l'app store de Nextcloud est instable .

Étape 1 : Se connecter au serveur Nextcloud
Ouvre un terminal et connecte-toi en SSH à ta VM nextcloud :

bash
vagrant ssh nextcloud
Étape 2 : Télécharger et installer le plugin
Exécute les commandes suivantes pour télécharger le plugin ONLYOFFICE et le placer dans le bon répertoire :

bash
# Aller dans le dossier des applications de Nextcloud
cd /var/www/html/nextcloud/apps

# Télécharger la dernière version du connecteur ONLYOFFICE
sudo wget https://github.com/ONLYOFFICE/onlyoffice-nextcloud/releases/latest/download/onlyoffice.tar.gz

# Extraire l'archive
sudo tar -xzvf onlyoffice.tar.gz

# Nettoyer l'archive
sudo rm onlyoffice.tar.gz
Étape 3 : Corriger les permissions
C'est une étape cruciale pour que Nextcloud reconnaisse l'application .

bash
sudo chown -R www-data:www-data /var/www/html/nextcloud/apps/onlyoffice
sudo chmod -R 755 /var/www/html/nextcloud/apps/onlyoffice
Étape 4 : Activer l'application
Maintenant, retourne sur l'interface web de Nextcloud.

Va dans Paramètres > Applications.

Clique sur la catégorie "Applications désactivées" (ou "Disabled apps").

Tu devrais y voir ONLYOFFICE. Il ne te reste plus qu'à cliquer sur "Activer".
